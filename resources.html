TropiTracker
Resources
Tropi•Tracker only uses information from the National Hurricane Center, ensuring that all data is accurate and reliable.

All data is fetched from official NHC RSS feeds. These feeds are all public and free to use. If you would like to use them in your own application, refer to this link.

Alternatively, Tropi•Tracker also has a public API that is open-source & free to use. However, it is currently unfinished and may not be as reliable as opposed to the National Hurricane Center's RSS feeds.
If you would like to use this API in your own applications, refer to api.tropitracker.com or visit the GitHub page.
